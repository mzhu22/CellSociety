Detailed Description

CASettings is a data structure that holds all info read from the XML file, including grid size, simulation type, and parameters. CASettings serves to encapsulate this data and pass it from XMLReader to the CALoop, where it is accessed and used to construct the visualization.

Though CASettings is a passive data object, its existence allows XML file-reading to be put into its own specialized class.
